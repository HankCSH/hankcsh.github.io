# HKIBIM BIM Automation Arena 2023 - Honourable Mention

HKIBIM hosted its first **BIM Automation Arena** to showcase innovative digital solutions advancing construction project management in design, visualization, collaboration, and data exchange.

## Project: Deep auto Modeller

**"Deep auto Modeller: A cross-visual programming tool for BIM data and deep learning"**

This project applies deep learning to **automate BIM detailing**, streamlining a traditionally manual process.

**[More details](https://www.hkibim.org/index.php?mid=53)**
